# 🔧 Scripts Toolkit

A collection of practical PowerShell scripts used for cloud automation and DevOps workflows.

These scripts are designed to streamline everyday tasks when working with Microsoft Azure.

## 📂 Included Scripts

### ✅ Check-AzureLogin.ps1
Checks if you're currently authenticated with the Azure CLI.
- **Use Case:** Before running any automation that depends on Azure authentication.
- **Command Used:** `az account show`

### ✅ List-AzureResourceGroups.ps1
Lists all Azure Resource Groups in the current subscription in a table format.
- **Use Case:** Quickly view what's provisioned.
- **Command Used:** `az group list --output table`
